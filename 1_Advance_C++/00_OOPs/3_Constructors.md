### Constructor

This is an especial function that is used to initilize the member variables during object creation.

#### 1. Default Constructor

A special constructor automatically generated by class if some constructor is not defined by the programmer.

> Members should be public.
> Function name should be same as class name.

#### 2. Parameterized Constructor

A Special constructor that accepts one or more parameter during object creation.


> There can be more than one constructor in any class

```cpp

class Strategy
{
    public :
        int order-qyt;
        double price;
        string user-name;

        Strategy( int qty , double prc , string name)
        {
            order-qty = qty;
            price = prc;
            user-name = name;
        }
};

int main()
{
    Strategy S(10 ,12.1, "Test");
    return 0;
}
```
